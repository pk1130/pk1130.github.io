Python | Scikit-Learn | Data Science | Regression

Data Classification Challenge

This python script uses a Logistic Regression model to predict whether a credit card customer will make his bill payment on time the following month or not. It uses Python's scikit-learn library to create data pre-processing pipelines and pass that into the regression model to make predictions. It produces a .csv file with a column filled with 0s and 1s, where 0s indicate failure to pay and 1s indicate correct payment. This model ranked in the top 30% of all submissions to the Kaggle challenge.

As part of this course, I built a complete compiler for a language called Small-C (involving basic syntax and operations derived from C) in OCaml, that consisted of a syntax parser (both lexer and parser) and an interpreter based on a context-free grammar provided for the same.
